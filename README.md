# 百家乐走势分析 微信小程序

## 项目结构

```
baccarat-mp/
├── app.js              # 全局逻辑 & 数据管理
├── app.json            # 页面路由 & TabBar 配置
├── app.wxss            # 全局样式
├── project.config.json # 微信开发者工具配置
├── pages/
│   ├── index/          # 主页：走势图 + 策略分析 + 拍照识别
│   └── history/        # 历史页：多靴记录汇总
└── utils/
    ├── analysis.js     # 大路算法 & 策略引擎
    └── imageOcr.js     # Claude Vision 图片识别
```

## 功能说明

### 1. 走势记录
- 点击【庄】【闲】【和】按钮记录每把结果
- 支持撤销最后一把
- 支持开新靴（自动按靴保存）

### 2. 大路走势图
- 按列排列，庄红闲蓝和绿
- 和局显示角标数字
- 横向滚动，支持长靴记录

### 3. 策略分析（4种）
| 策略 | 触发条件 |
|------|----------|
| 顺势策略 | 连续3把以上同方 |
| 剁策略 | 最近6把交替出现 |
| 热庄/热闲 | 近10把偏差2把以上 |
| 大势策略 | 15把后庄/闲比例>60% |

### 4. 📷 拍照识别（AI驱动）
使用 Claude Vision API 识别以下图片类型：
- 赌场屏幕上的珠盘路截图
- 手写或打印的记分纸
- 其他百家乐App的走势图截图

#### 配置步骤
1. 点击【📷 拍照识别】按钮
2. 首次使用会弹出 API Key 配置框
3. 前往 https://console.anthropic.com 获取 API Key
4. 输入 Key（格式：sk-ant-...）后保存
5. 拍照或从相册选择走势图
6. AI 识别完成后确认导入

## 部署步骤

1. 打开微信开发者工具
2. 导入本项目目录
3. 在 `project.config.json` 中填入你的 AppID
4. 在 `app.json` 的 tabBar 中补充 icon 图片（或删除 tabBar 改用顶部导航）
5. 在微信公众平台配置域名白名单：`api.anthropic.com`
6. 上传并发布

## 注意事项

- API Key 存储在本地，不上传至任何服务器
- 百家乐每把独立事件，走势分析仅供参考
- 请在合法场所理性游玩
